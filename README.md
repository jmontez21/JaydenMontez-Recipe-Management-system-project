# Recipe Management System
**Author:** Jayden Montez

## Purpose
This is a web application for managing recipes. Users can add new recipes with details like name, description, preparation time, and category. Users can also view all recipes and edit existing recipes.

## File Locations

### Root Directory (project5/)
- **server.js** - Express server file
- **database.sql** - Database creation script
- **package.json** - Node.js dependencies
- **README.md** - This file

### Public Directory (project5/public/)
- **index.html** - Home page
- **recipes.html** - Recipes page
- **styles.css** - Stylesheet
- **app.js** - Client-side JavaScript for index.html
- **recipes.js** - Client-side JavaScript for recipes.html
